# DocuMind

An AI-powered Document Intelligence Platform built using Retrieval-Augmented Generation (RAG).

## Features

- PDF Text Extraction using PyMuPDF
- Intelligent Text Chunking using LangChain
- Embedding Generation using Sentence Transformers
- Vector Storage using ChromaDB
- Semantic Search and Retrieval

## Tech Stack

- Python
- PyMuPDF
- LangChain
- Sentence Transformers
- ChromaDB

## Current Pipeline

PDF
↓
Text Extraction
↓
Chunking
↓
Embeddings
↓
ChromaDB
↓
Question
↓
Semantic Retrieval

## Example Query

Question:

What is metadata?

Retrieved Result:

Metadata (also called the data dictionary) is data about data.
